# Sistema de Préstamos - UNAB

Proyecto desarrollado para el ramo **Introducción a la Programación**.

## Descripción

Sistema de consola desarrollado en Python para gestionar préstamos y devoluciones de materiales, herramientas y equipos en una escuela.

## Funcionalidades

- Registro de alumnos
- Registro de docentes
- Registro de materiales
- Registro de préstamos
- Registro de devoluciones
- Validación de RUT por módulo 11
- Normalización de RUT
- Validación de email
- Validación de datos numéricos
- Reportes de personas
- Reportes de materiales
- Reportes de préstamos
- Reportes de devoluciones

## Archivos principales

- `menu.py`: contiene el menú principal del sistema.
- `SistemaPrestamos.py`: contiene la lógica principal del sistema.
- `modelos.py`: contiene las clases del sistema.

## Ejecución desde Python

```bash
python menu.py
```
